# Carrie Tennant Website

Premium static website concept for Carrie Tennant, Canadian conductor, educator, mentor, speaker, and youth choir specialist.

## Local Preview

```bash
python3 -m http.server 8791
```

Then open `http://127.0.0.1:8791/`.

## Files

- `index.html` - site content and structure
- `styles.css` - responsive visual design
- `script.js` - mobile navigation and sticky header behavior
